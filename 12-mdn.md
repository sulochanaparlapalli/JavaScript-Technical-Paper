# Searching MDN Documentation

## Definition

MDN (Mozilla Developer Network) is the **official and most trusted documentation source** for JavaScript and web technologies.

---

## Why Use MDN?

MDN provides complete and reliable information such as:

- Syntax
- Parameters
- Return Values
- Browser Support
- Real-world examples

---

## Example Search

Search:

```txt
mdn array map
```

You will find:

- Method syntax
- Examples
- Edge cases

---

## Example

```js
const numbers = [1, 2, 3];

const doubled = numbers.map(num => num * 2);

console.log(doubled);
```

Output:

```txt
[2, 4, 6]
```

---

## What MDN Helps You Understand

For map(), MDN will show:

- How the method works
- Callback function details
- Return value
- Edge cases
- Compatibility across browsers

---

## Key Points

- MDN is the most reliable JavaScript documentation
- Always prefer MDN over random blogs
- Learn to read official documentation instead of memorizing everything
- Understanding documentation is a key developer skill
